# Student Management System

A Full Stack Web Application to manage student records.

## Technologies Used
- React.js
- Python
- SQLite
- HTML5
- CSS3
- REST API

## Features
- Add Student
- Update Student
- Delete Student
- View Students
- Alphabetical Sorting
- Responsive UI

## Project Architecture
React Frontend → Python Backend → SQLite Database

## Run the Project

Backend:
python server.py

Frontend:
npm start
